# MagickPlot-RubyPlot
MagickPlot-RubyPlot is Matplotlib for Ruby.

# Testing
Installation can be done with ``` bundle exec rake install ```     
Run all the tests with ``` bundle exec rake spec ```       
Test the code in pry with ```bundle exec rake pry  ```
